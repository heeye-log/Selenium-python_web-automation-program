### Troubleshooting
- Selenium 설치하는 것 부터 난관에 빠져 있었다. 알고 보니 OS별로 다른 부분도 있었지만 사이트가 업데이트 되었거나 에러 코드를 하나씩 구글링하여 수정된 명령어로 치고나서야 제대로 작동했다. 
혹시나 예방하기 위해서 블로그에 [설치 방법](https://velog.io/@heehe/selenium-chrome-driver-%EC%84%A4%EC%B9%98) 기입하였다.
